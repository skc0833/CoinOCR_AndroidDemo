## Build
- Android Studo 에서 File / Open... 메뉴를 클릭 후, CoinOCR_AndroidDemo 폴더를 오픈한다.
- Sync Project with Gradle Files 클릭
- OpenCV 빌드 에러가 발생할 경우, Build / Clean Project 를 하게 되면 이 시점에 app\OpenCV 폴더를 내려받으면서 다시 Gradle Sync 하게 되면 성공함
